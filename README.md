## My Dotfiles

# Homebrew
I'm using `homebrew-bundle` to version control the packages I use (https://github.com/Homebrew/homebrew-bundle)

To install:

```
$ brew bundle
```

# VSCode

To sync settings from dotfiles, you'll need to synlink. Directions can be found here: https://pawelgrzybek.com/sync-vscode-settings-and-snippets-via-dotfiles-on-github/
